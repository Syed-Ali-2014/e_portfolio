Unit 7: Hypothesis Testing Using Excel and Summary Measures

In this unit, I learned how to perform hypothesis testing using Excel's built-in functions. Below are the details of the hypothesis tests conducted, along with the calculations and results.

Hypothesis Testing

Here are the key hypothesis tests I conducted using the t-Test functions in Excel:

Hypothesis t-Statistic p-Value Conclusion
Test of Difference between Con1 and Con2 2.875 0.018 Reject H0
Test of Difference between Filter Agent 1 and Filter Agent 2 (One-tailed) 1.940 0.03 Reject H0
Test of Population Mean Income for Males vs Females 3.270 0.00071 Reject H0
Findings:
  • The test between Con1 and Con2 showed significant differences, with a p-value of 0.018, rejecting the null hypothesis.
  • Filter Agent 1 is significantly more effective than Filter Agent 2, as indicated by the rejection of H0 with a p-value of 0.03.
  • Males earn significantly more than females in the given population, with a p-value of 0.00071, showing strong evidence to reject the null hypothesis.
